# 🍽️ Планувальник меню сім'ї + AI Кухар

Веб-додаток для планування сімейного меню з AI генератором рецептів.

## ✨ Можливості

- 📋 **Планування готування** на 3-4 дні наперед
- 🍴 **Каталог страв** з категоріями (білки, гарніри, овочі, десерти)
- ✅ **Облік готових страв** в холодильнику
- 🛒 **Автоматичний список покупок** з планів готування
- 🤖 **AI генератор рецептів** - створює нові рецепти на запит

## 🚀 Технології

- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Backend**: Node.js, Express.js
- **База даних**: SQLite
- **AI**: Демо версія (готова для інтеграції з OpenAI API)

## 📱 Як використовувати

1. **Планувальник** - додай страви які збираєшся готувати
2. **AI Кухар** - згенеруй нові рецепти за запитом
3. **Каталог страв** - переглядай збережені рецепти за категоріями
4. **Готові страви** - відслідковуй що вже приготовлено
5. **Список покупок** - генеруй автоматично з планів

## 🎯 Особливості

- Адаптивний дизайн для мобільних пристроїв
- Швидкі кнопки для додавання страв
- Статистика частоти готування
- Автоматичні пропозиції рецептів

## 🔧 Установка

```bash
npm install
npm start
```

Додаток буде доступний на `http://localhost:3000`

## 📝 API

- `GET /api/dishes` - отримати страви
- `POST /api/dishes` - додати страву
- `GET /api/cooking-plans` - отримати плани готування
- `POST /api/cooking-plans` - додати план
- `GET /api/shopping-list` - отримати список покупок

## 🎨 Скріншоти
